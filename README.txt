INTRODUCTION

- Reversi se joue sur un plateau unicolore de 8 x 8 cases. 

- Les colonnes sont numérotées de gauche à droite par les lettres A à H ; les lignes sont numérotées de haut en bas par les chiffres de 1 à 8. 

- Les joueurs disposent de 64 pions bicolores (une face noire et une face blanche).

- Le jeux fini quand l'un des deux joueurs n'a aucune case possible a jouer ou il n'y a pas aucune case nulle, le resultat base sur le nombre de pion

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

COMMENT JOUER?

- En début de partie, pour les quatre premiers coups, chacun place alternativement un pion montrant la face de sa couleur sur une case centrale (D4, D5, E5, E5).

- Ensuite, chacun à son tour place un pion de sa couleur sur une case de son choix, à condition que cette case soit vide et située à l’extrémité d’un alignement de pions adverses contigus et dont l’autre extrémité est déjà occupée par un de ces propres pions (l’alignement considéré peut être orthogonal ou diagonal : 8 alignements possibles)

- Si un joueur ne peut pas poser de pion en respectant ces conditions, la main passe. 

- Chaque joueur peut passer la main en cliquant le bouton "Passer" si et seulement s'il existe au moins une case possible mais le joueur ne peut pas la trouver.

- Le logiciel vous permet aussi de sauvegerder le jeux en fichier xml et de ouvrir le fichier xml pour continuer a joueur avec les 2 boutons "Sauvegarder" et "Ouvrir".

- Bouton "Rejoueur" va effacer tous les pions sur la grille pour rejouer.

- Pour facilement a jouer, vous pouvez cocher "Aide" qui va colorer les cases valides.
Les cases valides affichées:
 	+ en gris pour joueur NOIR.
	+ en orange pour joueur BLANC.
